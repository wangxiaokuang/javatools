 **版本号说明** 

- 主版本号.次版本号.修正版本号
    1. 当项目进行局部微调或修复bug时,主版本号与次版本号均不变，修正版本号加一
    2. 当项目进行增加功能时，主版本号不变，次版本号加一，并将修正版本号清零
    3. 当项目进行重大修改或局部修改累计次数较多时，主版本号加一，并将次版本号与修正版本号清零
